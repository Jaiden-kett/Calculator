# Python Calculator

A command-line calculator written in Python that performs multiple arithmetic operations on a list of numbers.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Remainder
- Power

## How to Run

1. Clone the repository
2. Run the program

python Calculator.py
